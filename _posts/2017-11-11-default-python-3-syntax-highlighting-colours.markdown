### These Are The Default Python 3 Syntax Highlighting Colours

*   Comments: #dd0000
*   Keywords: #ff7700
*   Definitions: #0000ff
*   Normal Text: #000000
*   String: #00aa00
*   Selected Text: #000000
*   Found Text: #ffffff
*   Builtins: #900090
*   Editor Breakpoint: #000000
*   Shell Error Text: #000000
*   Cursor: #000000
*   Shell Normal Text: #770000
*   Shell Stdout: #0000ff
*   Shell Stderr: #ff0000
